# 快速入門指南

## 5 分鐘開始使用習慣追蹤器

### 1️⃣ 克隆專案

```bash
git clone https://github.com/ichun0512/Joanne-tracker.git
cd Joanne-tracker
```

### 2️⃣ 安裝依賴

```bash
npm install
```

### 3️⃣ 啟動開發伺服器

```bash
npm run dev
```

### 4️⃣ 開啟瀏覽器

訪問 [http://localhost:3000](http://localhost:3000)

---

## 首次使用教學

### 步驟 1: 註冊帳號

1. 點擊「還沒有帳號？立即註冊」
2. 填寫：
   - 姓名：你的名字
   - 電子郵件：用於登入
   - 密碼：至少 6 個字元

### 步驟 2: 建立第一個習慣

1. 登入後，點擊「新增習慣」按鈕
2. 填寫習慣資訊：
   ```
   名稱：晨跑
   描述：每天早上跑步 30 分鐘
   頻率：每日
   目標次數：1
   顏色：選擇你喜歡的顏色
   ```
3. 點擊「建立習慣」

### 步驟 3: 開始打卡

1. 在「今日習慣」頁面
2. 找到你剛建立的習慣
3. 點擊右上角的 ✓ 按鈕完成打卡
4. （可選）點擊「備註」添加今天的心得

### 步驟 4: 查看統計

1. 點擊導覽列的「統計」按鈕
2. 查看：
   - 總習慣數
   - 總打卡次數
   - 最長連續天數
   - 平均完成率

### 步驟 5: 使用月曆視圖

1. 點擊「月曆」按鈕
2. 查看每天的打卡記錄（用彩色點點標示）
3. 點擊任意日期進行補打卡

## 常見使用場景

### 📝 建立多個習慣

```
建議習慣組合：
- 早晨：冥想 10 分鐘
- 上午：閱讀 30 分鐘
- 下午：運動 1 小時
- 晚上：寫日記
```

### 🔄 每週習慣追蹤

```
設定頻率：每週
目標次數：3
範例：每週健身 3 次
```

### 📊 查看進度

1. 「統計」頁面查看整體表現
2. 每個習慣都有詳細的：
   - 目前連續天數
   - 最長連續天數
   - 總打卡數
   - 完成率（帶進度條）

### 🌙 切換深色模式

點擊右上角的月亮/太陽圖示即可切換主題

## 快速鍵提示

- `註冊` → 首次使用
- `+ 新增習慣` → 快速建立習慣
- `✓` → 快速打卡
- `備註` → 記錄心得
- `編輯` → 修改習慣資訊
- `月曆` → 查看歷史記錄

## 資料說明

- 所有資料存儲在瀏覽器的 LocalStorage
- 不需要網路連線即可使用
- 資料僅存在本地，保護隱私
- 清除瀏覽器資料會刪除所有記錄（請注意備份）
